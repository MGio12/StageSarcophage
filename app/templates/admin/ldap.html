Variables d'environnement LDAP
Variable Description Exemple
LDAP_ENABLED Activer l'authentification LDAP true
LDAP_HOST Adresse du serveur LDAP ldap.example.com
LDAP_PORT Port (389 par defaut, 636 pour SSL) 389
LDAP_USE_SSL Utiliser SSL/TLS false
LDAP_BASE_DN DN de base pour les recherches DC=example,DC=com
LDAP_BIND_DN DN du compte de service CN=service,OU=Service,DC=example,DC=com
LDAP_BIND_PASSWORD Mot de passe du compte de service ***
LDAP_USER_FILTER Filtre de recherche utilisateur (sAMAccountName={username})
LDAP_DEFAULT_ROLE Role par defaut pour les nouveaux utilisateurs LDAP lecteur
LDAP_SYNC_GROUPS Activer la synchronisation des groupes true
LDAP_GROUP_MAPPING Mapping groupes AD -> roles CN=Admins,DC=...:admin;CN=Users,DC=...:lecteur
